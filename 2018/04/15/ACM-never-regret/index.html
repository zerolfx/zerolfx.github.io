Posted on 2018-04-15 | In ACM |

若干年后，回首如今曾为 ACM 奋斗的自己，一定不会后悔。

前言

题目主要选自洛谷，因为自带按照赞数排序的题解，A 完一道题后一定要看看别人的代码或算法是不是更为简洁、优美（一定要多看几份，排在前面的不一定写得好）。

尽可能按照难度顺序排列，保证（几乎）所有题都有题解。

不那么简单的 DP

DP 要求无后效性，和最优子结构。参考：https://zh.wikipedia.org/wiki/%E5%8A%A8%E6%80%81%E8%A7%84%E5%88%92

DP 题的特点是当你看了题解的状态转移方程，可能一下子就明白了，但偏偏想不到。DP 的形式太多了，所以重要的是理解 DP 的思想。

  • 枚举两行/列，将二维问题转化为一维，然后就是最大连续子序列和 https://www.luogu.org/problemnew/show/P1719
  • https://www.luogu.org/problemnew/show/P2690
  • https://www.hackerrank.com/challenges/fair-cut/problem
