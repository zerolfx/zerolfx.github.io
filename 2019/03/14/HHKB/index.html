【测评】HHKB Pro BT

Posted on 2019-03-14 Edited on 2019-08-09 In 测评 Disqus:

算是对于网上的评测的一些基于个人使用体验的补充。

缺点

  • 线只能用来供电但不能当有线键盘使用，这个设定就很垃圾了，无话可说。（Filco 双模键盘就支持蓝牙或者有线）
  • 虽然蓝牙支持与四个设备配对，但不能无缝切换，切换蓝牙相当于断开并重新配对，缓慢且需要额外的操作。（反观 Filco 就可以同时连接并切换，真正的多设备）
  • 如果蓝牙不稳定的话会抽搐（比如一个键不断被重复以及高延迟），不过这个应该是蓝牙设备的通病，由于出现频率很低，所以姑且不影响使用，但是一旦出现就会很心烦。
  • 配列键位。既是优点也是缺点，由于 Fn 要用组合键实现，那么有些快捷键，比如 Ctrl-Shift-F9 就需要四个键了，会有一些别扭。

优点

  • 配列特殊。退格键位置好评，这个只能通过改变键盘配列来做到，光靠键盘映射没法达到一样的效果。
  • 体积小，重量轻，便携。可以放在笔记本上面使用。
  • 触感和手感都不错。
  • 逼格高。请配合 Emacs 或者 Vim 使用更佳。

建议

  • 键盘配列特殊，如果觉得不能接受就千万不要买了再适应（尤其看清楚方向键的组合方式）。或者不妨先修改映射适应一段时间。
  • 光是喜欢配列的话也还有别的选择。
  • 静电容，敲击的声音还是挺响的。（大概红轴水平？）
  • 六个跳线开关，个人建议选 Mac 模式，其他开关全部 up。因为 Mac 模式多了几个多媒体键，而且在 Windows 下也支持，反倒是 Linux 不支持。原本只有右侧有 Fn，这会导致一些单手快捷键变成双手快捷键（比如 F1），所以在左侧也搞出一个 Fn 是挺有必要的。
  • 从 HHKB 键盘换到普通键盘也不会产生什么不适应的问题，所以，没什么问题。
  • 如果不喜欢诡异布局的小键盘的话（如果不考虑便携性的话，买小键盘实在是没什么必要），可以看看 Realforce 的静电容键盘，手感也挺不错的（在 China Joy 上试过）。
